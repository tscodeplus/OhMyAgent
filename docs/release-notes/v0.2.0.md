## 🚀 Features

- **Multi-provider AI Agent Gateway** — Seamless messaging across Feishu/Lark, Telegram, WeChat, QQ, Cron, and more
- **LLM-powered Tool Execution** — Built-in tools (web_search, computer-use, memory) with approval workflows and policy engine
- **Memory System** — SQLite + sqlite-vec vector search with entity extraction and contextual memory
- **Skills System** — Trigger-word/command-based skill matching with automatic prompt and tool policy injection

## 🖥️ Desktop App

- Electron desktop client with system tray and auto-start support
- Connect to remote gateways; manage local and remote files from the desktop
- Windows (NSIS installer), macOS Intel + Apple Silicon (Universal DMG), Linux (AppImage + deb)

## 🕸️ WebUI

- Chat and execute tasks without configuring any IM channel
- Centrally manage model providers, agents, web search, memory system, and gateway settings

## 🚀 功能特性

- **多供应商 AI Agent 网关** — 统一接入飞书/Lark、Telegram、微信、QQ、Cron 等多渠道消息
- **LLM 驱动的工具执行** — 内置 web_search、computer-use、memory 等工具，支持审批流和策略控制
- **记忆系统** — SQLite + sqlite-vec 向量检索引擎，支持实体提取和上下文记忆
- **技能系统** — 通过触发词/命令进行技能匹配，自动注入 prompt 和工具策略

## 🖥️ 桌面应用

- Electron 桌面客户端，支持常驻系统托盘和开机自启动
- 支持连接远程网关、操作桌面端本地文件和远程文件
- 提供 Windows (NSIS)、macOS Intel + Apple Silicon (Universal DMG)、Linux (AppImage + deb) 安装包

## 🕸️ 管理面板

- 无需配置 IM 即可进行对话和执行任务
- 方便地管理模型供应商、Agent、网络搜索、记忆系统、网关等配置
