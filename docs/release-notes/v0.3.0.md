## 🚀 New Features

- **Agent Template Library** — Browse and import agent configs from the new `agency-agents` template library directly from the WebUI. Discover community-built agent personas for coding, writing, analysis, and more — one click to add them to your setup
- **WebUI OAuth Login** — Sign in to Claude Pro/Max, ChatGPT Plus/Pro, or GitHub Copilot directly from the WebUI with interactive OAuth prompts. No more copying tokens manually — the browser handles the OAuth flow end-to-end

## 🛠 Improvements

### Subscription Login

- Moved subscription management into **Models > Providers** section — it now lives alongside your API keys and custom providers where it belongs
- **Compact UI** — Subscription cards are now collapsed by default with a cleaner layout
- Renamed to **Subscription Login** for clarity
- **Better error handling** — OAuth login errors are sanitized and displayed as toasts instead of inline text; toast duration extended to 8 seconds so errors are readable

### Sessions

- **Auto-numbered titles** — Default session names now include an incrementing number (e.g. "New Chat 1", "New Chat 2") to avoid duplicate titles
- **Fixed timestamps** — Timestamps with `.0` suffix (e.g. `172800000.0`) no longer incorrectly display as "just now"

### Template Browser

- Header is now sticky so controls remain visible when scrolling through a long template list
- Left panel list has a scrollbar for easier navigation

## 🚀 新功能

- **Agent 模板库** — 通过 WebUI 浏览和导入 `agency-agents` 模板库中的 Agent 配置。一键添加社区构建的编程、写作、分析等各类 Agent 角色
- **WebUI OAuth 登录** — 直接在 WebUI 中通过交互式 OAuth 提示登录 Claude Pro/Max、ChatGPT Plus/Pro 或 GitHub Copilot。浏览器端完成整个 OAuth 流程，无需手动复制 token

## 🛠 改进

### 订阅登录

- 订阅管理移动到 **AI 模型 > 供应商** 页面，与 API Key 和自定义供应商放在一起
- **紧凑 UI** — 订阅卡片默认折叠，布局更简洁
- 更名为 **订阅登录** 以明确功能
- **错误处理优化** — OAuth 登录错误经过过滤后以 toast 显示，去除内联错误消息；toast 展示时间延长至 8 秒，确保错误信息可读

### 会话

- **标题自动编号** — 默认会话名称包含递增编号（如"新会话 1"、"新会话 2"），避免列表中重复标题
- **修复时间戳** — `.0` 后缀的时间戳（如 `172800000.0`）不再错误显示为 "just now"

### 模板浏览器

- 顶部标题栏固定，滚动长列表时控制项保持可见
- 左侧列表添加滚动条，方便浏览
