v0.9.1 fixes self-update from WebUI with real-time progress display.

## 🐛 Bug Fixes

- **WebUI self-update** — `deploy-termux.sh` now includes the `.git` directory in tarballs and switches the remote to HTTPS so `git pull` works for future updates. A new `/api/system/update-status` endpoint reports update progress, and the WebUI polls for status after triggering a self-update, displaying real-time progress instead of a static spinner.
- **Tar exclude narrowed** — the deploy tarball generator no longer strips every `data` directory in the source tree; only `./data` (the project root data directory) is excluded. Also excludes `.codegraph` from tarballs.
