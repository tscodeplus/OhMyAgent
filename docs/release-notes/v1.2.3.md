v1.2.3 improves the WebUI update notification experience by rendering release notes with markdown formatting.

## 🎨 WebUI Update Notifications

- **Markdown rendering** — Toast component now renders update release notes via `ReactMarkdown`, so formatting (bold, lists, links) displays properly instead of as raw markdown text
- **Desktop compatibility** — Electron desktop path continues using plain-text truncation since update notes arrive as HTML format
