v1.1.1 is a bug-fix release focused on macOS desktop app distribution improvements.

## 🍎 macOS Desktop

- **Auto-update zip target** — added zip build target in electron-builder config for macOS auto-update support
- **DMG icons centered** — explicit `dmg.window` layout in electron-builder.yml so app icons are properly centered in the DMG installer window
- **Unsigned build redirect** — auto-updater now detects unsigned macOS builds and redirects to the manual download page instead of failing silently
- **quitAndInstall fix** — bypass `closeToTray` setting during `quitAndInstall()` so the update install flow properly terminates the app

***

v1.1.1 是面向 macOS 桌面端分发优化的修复版本。

## 🍎 macOS 桌面端

- **自动更新 zip 目标** — electron-builder 配置中新增 zip 构建目标，支持 macOS 自动更新
- **DMG 图标居中** — 显式配置 `dmg.window` 布局，应用图标在 DMG 安装窗口中正确居中
- **未签名构建重定向** — 自动更新器检测未签名的 macOS 构建，重定向到手动下载页面而非静默失败
- **quitAndInstall 修复** — 更新安装时绕过 `closeToTray` 设置，确保更新流程正常终止应用
