v0.9.2 fixes self-update reliability issues and an image preview bug.

## 🐛 Bug Fixes

- **Self-update reliability** — multiple fixes for the WebUI self-update workflow: handle local modifications with `git stash` before pull, add fetch timeout, cache app version at startup to prevent false "up to date" message after partial update, prevent concurrent update script spawning, clear polling interval and toasted version ref on cancel to prevent state corruption, make success and error toasts persistent, add page refresh hint on completion
- **Image preview** — fixed image preview failure caused by `/dl/` route mismatch and Content-Disposition header handling
