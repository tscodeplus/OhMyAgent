v0.6.1 refines the skill trigger system with fuzzy matching for multi-word triggers, removal of overly broad trigger keywords, and improved multi-skill activation display.

## 🛠 Improvements

- **Fuzzy matching for multi-word ASCII triggers** — triggers like `generate image` now match `generate an image`, `generate a nice image`, etc. (up to 3 filler words between each word pair), making skill activation more natural without needing to type exact trigger phrases
- **Broad trigger keyword cleanup** — removed ambiguous single-word triggers from researcher (search, look up, find information, 搜索, 查找) and image-generator (插图, 配图, 修改图片, 改图) to reduce unintended skill activation
- **Multi-skill activation display** — when several skills match a message, the activation badge now shows all matched skill names joined by ` | ` with an adaptive label ("技能激活" for one, "技能合并激活" for multiple)

v0.6.1 优化了技能触发系统：多词 ASCII 触发词支持模糊匹配、移除过于宽泛的触发关键词、改进多技能激活展示。

## 🛠 改进

- **多词 ASCII 触发词模糊匹配** — `generate image` 等触发词现在可匹配 `generate an image`、`generate a nice image` 等（每对词之间允许 0-3 个填充词），让技能激活更自然，无需输入精确的触发短语
- **宽泛触发关键词清理** — 移除了 researcher 的模糊触发词（search, look up, find information, 搜索, 查找）和 image-generator 的模糊触发词（插图, 配图, 修改图片, 改图），减少非预期的技能激活
- **多技能激活展示** — 当多个技能同时匹配一条消息时，激活标签显示所有匹配的技能名称（用 ` | ` 分隔），标签文字自动切换（单个→"技能激活"，多个→"技能合并激活"）
