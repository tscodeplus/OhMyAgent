v0.5.3 fixes several desktop progress window issues — unresponsive buttons, focus loss, and double-click jump — and adds a cancel button to the About page update flow.

## 🚀 New Features

### Cancel Button in About Page

The About page update flow now includes a **cancel button**, allowing you to abort a download in progress directly from the WebUI or Electron window.

## 🛠 Improvements

- **Progress window button fixes** — buttons in the native progress window no longer become unresponsive after focus loss; window focus issues resolved
- **Double-click jump** — fixed an issue where double-clicking update buttons could cause unexpected navigation jumps
- **Diagnostic log verbosity** — slimmed down `electron-diag.log` output to reduce noise during normal operation

---

v0.5.3 修复了桌面端进度窗口的几个问题 — 按钮无响应、焦点丢失和双击跳转，并为关于页面更新流程添加了取消按钮。

## 🚀 新功能

### 关于页面取消按钮

关于页面的更新流程现在包含**取消按钮**，可从 WebUI 或 Electron 窗口直接中止正在进行的下载。

## 🛠 改进

- **进度窗口按钮修复** — 原生进度窗口中的按钮在焦点丢失后不再无响应；窗口焦点问题已解决
- **双击跳转** — 修复了双击更新按钮可能导致意外导航跳转的问题
- **诊断日志精简** — 减少 `electron-diag.log` 输出，降低正常运行时的日志噪音
