v0.5.2 overhauls the desktop update flow with download progress visualization, a native progress window, multi-platform update support in WebUI, and improved error handling throughout.

## 🚀 New Features

### Download Progress Visualization

- **Progress bar in About page** — the "Check for Updates" flow now shows a real download progress bar instead of a static spinner
- **Native progress window** — Electron auto-update shows a dedicated progress window with download percentage, speed, and estimated time
- **WebUI multi-platform update** — the `perform-update` endpoint now detects Windows/macOS/Linux and downloads the correct platform artifact

## 🛠 Improvements

- **Progress window UX** — cleaner layout, proper button ordering, and scrollbar support for long messages
- **Error handling** — graceful degradation when `app-update.yml` is missing (portable builds); better error logging throughout the update flow
- **Toast notifications** — release notes in update toasts now have scrollbars for overflow content and HTML tags are stripped for clean display
- **Portable build** — added `--publish never` to prevent CI from detecting the portable build as a published release

---

v0.5.2 全面改进了桌面端更新流程：下载进度可视化、原生进度窗口、WebUI 多平台更新支持，以及全链路的错误处理优化。

## 🚀 新功能

### 下载进度可视化

- **关于页面进度条** — "检查更新"流程现在显示真实下载进度条，替代静态转圈
- **原生进度窗口** — Electron 自动更新展示专用进度窗口，包含下载百分比、速度和预估时间
- **WebUI 多平台更新** — `perform-update` 端点自动识别 Windows/macOS/Linux 并下载对应平台的安装包

## 🛠 改进

- **进度窗口体验** — 更清晰的布局、合理的按钮顺序、长消息支持滚动条
- **错误处理** — `app-update.yml` 缺失时优雅降级（便携版）；全链路改进错误日志
- **Toast 通知** — 更新提示中的 release notes 溢出时显示滚动条，HTML 标签自动清理
- **便携版构建** — 添加 `--publish never` 防止 CI 将便携版误识别为已发布版本
